Image Studio

    How to Use Image Studio

    Getting Started

    1. Register or log in to your account.

    2. Navigate to the home page to start generating images.

    Generating Images

    - Enter a creative prompt describing what you want.

    - Optionally, add a negative prompt to exclude elements.

    - Select resolution and number of images (up to 4).

    - Click "Generate Now" to create your images.

    Viewing and Downloading

    - Generated images appear in the gallery below the form.

    - Click an image to view it in a modal.

    - Use the "Download" button to save the image.

    Managing Your Account

    - Upgrade your membership for more features.

    - Check your generated images in the Gallery page.
